QUEST√ÉO 1)

Para comparar as m√©dias de produ√ß√£o de madeira de dois clones de eucalipto, vamos testar duas hip√≥teses, a primeira sendo, se as m√©dias das produc√µes dos dois clones s√£o iguais e a segunda se as m√©dias s√£o diferentes, e para isso usaremos o teste t, com um n√≠vel de signific√¢ncia de 5%;

$H_{0}:\text{a produ√ß√£o de madeira m√©dia dos dois clones s√£o iguais}$

$H_{1}:\text{a produ√ß√£o de madeira m√©dia de um clone √© diferente}$

{r}

clone_a <- c(13, 19, 31, 15, 20, 30, 22, 35, 18, 33)
clone_b <- c(31, 36, 39, 27, 26, 29, 45, 30, 28, 43)


dados <- data.frame(
  Produ√ß√£o = c(clone_a, clone_b),
  Clone = rep(c("A", "B"), each = 10)
)


Teste t para duas amostras independentes:

{r}
teste_t <- t.test(clone_a, clone_b, var.equal = TRUE)
print(teste_t)


Pelo teste t rejeitamos a hip√≥tese nula, uma vez que, as m√©dias de produ√ß√£o de madeira dos clones s√£o diferentes. Os valores encontrados de m√©dia para o clone A e para o clone B foram 23,6 e 33,4 metros c√∫bicos de madeira, respectivamente.

ANOVA:

{r}
anova_result <- aov(Produ√ß√£o ~ Clone, data = dados)

summary(anova_result)


Pela ANOVA, tamb√©m rejeitamos a hip√≥tese nula, uma vez que, as m√©dias de produ√ß√£o de medeira dos clones s√£o diferentes.

Verificando se t2 √© igual a F:

{r}
t_squared <- teste_t$statistic^2

f_value <- summary(anova_result)[[1]]["Clone", "F value"]

cat("t^2 =", t_squared, "\nF =", f_value, "\n")


Como observado, o valor t2 √© igual ao valor de F, o que confirma a igualdade entre os testes quanso se compara apenas dois tratamentos.

Conclus√µes

Como existe diferen√ßa significativa implica que as pr√°ticas de manejo ou sele√ß√£o do clone para produ√ß√£o florestal devem considerar as diferen√ßas observadas, possivelmente preferindo o clone com maior m√©dia de produ√ß√£o, sendo esse, o clone B.

Essa equival√™ncia do t2 com o teste F ocorre porque, com dois grupos, a soma de quadrados total pode ser dividida diretamente entre os quadrados entre os grupos (tratamentos) e os quadrados dentro dos grupos (erro), resultando na mesma f√≥rmula subjacente.

Quando h√° mais de dois grupos A ANOVA generaliza o conceito para comparar mais de dois tratamentos simultaneamente, avaliando se pelo menos um dos grupos difere dos outros. Nesse caso, o teste t deixa de ser aplic√°vel diretamente porque ele s√≥ compara dois grupos por vez. Para m√∫ltiplos tratamentos, usar o teste t repetidamente aumentaria o risco de erro tipo I (rejeitar ùêª0 quando ele √© verdadeiro).

Croqui de instala√ß√£o do experimento:

O experimento foi conduzido com 4 tratamentos (cores de papel√£o) e 6 repeti√ß√µes (posi√ß√µes aleat√≥rias). No croqui, cada c√©lula representa uma posi√ß√£o de um papel√£o no campo.A aloca√ß√£o foi feita aleatoriamente, garantindo que cada cor (tratamento) apare√ßa 6 vezes, minimizando vi√©s.

{r}
